# Elevatelab_Task_15

#  Project Overview
This project demonstrates a complete end-to-end Machine Learning pipeline using the Breast Cancer Dataset. The objective is to build a production-ready ML workflow that includes preprocessing, model training, evaluation, and model saving.


#  Dataset
- Dataset Used: Breast Cancer Dataset
- Target Variable: Diagnosis (Malignant = 1, Benign = 0)
- Features: Various tumor-related numerical measurements


#  Tools & Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab


#  ML Pipeline Workflow

1. Load Dataset  
2. Clean Data (remove ID column, handle missing values)  
3. Separate Features and Target  
4. Perform Train-Test Split  
5. Apply Standard Scaling  
6. Train Logistic Regression Model  
7. Evaluate Model (Accuracy, Precision, Recall, F1-Score)  
8. Save Trained Model as `.pkl` file  


#  What is an ML Pipeline?

An ML Pipeline automates the complete machine learning workflow including preprocessing and model training. It helps in reducing data leakage and makes deployment easier.


#  Model Evaluation Metrics

The model is evaluated using:

- Accuracy
- Precision
- Recall
- F1 Score
- Classification Report
